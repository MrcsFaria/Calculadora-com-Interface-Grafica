# Sobre o projeto

Este projeto é uma calculadora com interface gráfica construída com customtkinter e tkinter. A calculadora oferece funcionalidades básicas de cálculos, além de algumas funções matemáticas avançadas.
Fiz este projeto para praticar a criação de interfaces gráficas utilizando bibliotecas python.

# Tecnologias utilizadas
- Python
  
# Bibliotecas Python:
- customtkinter: Utilizada para criar a interface gráfica com componentes customizáveis.
- tkinter: Biblioteca padrão do Python para interfaces gráficas.
- math: Biblioteca padrão do Python para funções matemáticas.

# Como executar o projeto

Pré-requisitos: Python

```bash
# Clonar repositório
git clone Calculadora-com-Interface-Grafica

# Navegue até o diretório do projeto:
cd Calculadora-com-Interface-Grafica

# Instale as dependências necessárias
pip install customtkinter tkinter

# executar o projeto
python main.py
```
